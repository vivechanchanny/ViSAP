About ViSAP

Video is considered mainly as one way flow of communication wherein the interaction aspect is missing. ViSAP is a web based tool using which instructor can build pedagogical videos through video enrichments produced by adding annotations, sketches, questions, hot spot, whiteboard, tags and links which will provide additional information about the content in different forms and can be added with ease. It also gives instructor to have its own video store where enriched videos can be published for end user.

Visualization

Synchronized enrichments – Tool facilitates user to add enrichments which can be synchronized. Interactive enrichments – Option to add an interactive enrichment at a certain timestamp is given to user.

Collaboration

Multiple instructors can collaborate and create a single video with enrichment.

Video Snippet Merging

Snippets from different videos can be clipped and merged together to form a new video.

Assessments

ViSAP can include assessments support by providing templates for various question types.

Reusable Content Repository

ViSAP comes with a video repository where all video assets can be uploaded and published for future reference. This ensures a centralized repository of enriched videos and also facilitates easy discovery and reusability. Apart from this, the repository includes the standard features like categorizing each video for easy search.
